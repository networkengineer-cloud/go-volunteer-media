## Color Scheme

### Badges
- **Announcements**: Yellow/amber background (#fef3c7), brown text (#92400e)
- **Comments**: Blue background (#dbeafe), blue text (#1e40af)

### Status Indicators
- **Available**: Green (#10b981)
- **Foster**: Orange (#f59e0b)
- **Quarantine**: Red (#ef4444)
- **Archived**: Gray (#6b7280)

### Interactive Elements
- **Primary Button**: Teal (#0e6c55)
- **Secondary Button**: Gray with border
- **Hover**: Slightly darker shade + shadow
- **Focus**: 2px outline, 2px offset

## Interaction Patterns

### Tab Navigation
1. Click "Activity Feed" tab â†’ Shows activity feed
2. Click "Animals" tab â†’ Shows animals grid
3. Active tab has underline and bold text
4. Keyboard: Tab to focus, Enter to activate

### Filtering
1. Click filter dropdown â†’ Shows options
2. Select option â†’ Feed updates immediately
3. Maintains scroll position
4. Keyboard accessible

### Posting Announcement
1. Click "New Announcement" â†’ Modal opens
2. Fill in title and content (required)
3. Optionally upload image
4. Click "Post Announcement" â†’ Submits
5. Modal closes â†’ Feed refreshes
6. ESC key closes modal

### Infinite Scroll
1. Scroll to bottom â†’ Triggers load
2. "Loading more..." appears
3. New items append to feed
4. "End of feed" when done

### Viewing Images
1. Click image in activity item
2. Modal opens with full-size image
3. Click outside or ESC to close

### Navigation to Animal
1. Click animal thumbnail in comment
2. OR click "View animal" link
3. Navigates to animal detail page

## Accessibility Features

### Keyboard Navigation
- **Tab**: Move between interactive elements
- **Enter/Space**: Activate buttons/links
- **Escape**: Close modals
- **Arrow keys**: Navigate within lists

### Screen Reader Announcements
- "Loading activity feed" (aria-busy)
- "Loading more items" (aria-live)
- "You've reached the end of the feed" (role="status")
- "Announcement" or "Comment" for each item type

### Focus Management
- Visible focus rings (2px solid, 2px offset)
- Focus trapped in modals
- Focus returns to trigger after modal close
- Skip links available

## Responsive Breakpoints

### Mobile (< 640px)
- Single column layout
- Stacked buttons
- Tabs scroll horizontally if needed
- Cards full width

### Tablet (640px - 1024px)
- 2-column animal grid
- Side-by-side buttons
- Comfortable spacing

### Desktop (> 1024px)
- 3-4 column animal grid
- Centered content (max 1280px)
- Generous spacing
- Hover effects

## Loading States

### Initial Load
- Skeleton screens for 5 activity items
- Shimmer animation
- No content flash

### Loading More
- Spinner icon + "Loading more..."
- Appears after last item
- Smooth transition

### Empty States
- Icon + heading + description
- Call-to-action button when applicable
- Friendly, helpful tone

## Error Handling

### Network Errors
- Error icon + message
- "Try Again" button
- Preserve user input
- Don't clear forms

### Validation Errors
- Inline error messages
- Red border on invalid fields
- Specific guidance
- Disabled submit until valid

### Success Feedback
- Toast notification
- Green checkmark icon
- Auto-dismisses after 3s
- Non-intrusive

This visual guide demonstrates the thoughtful UX design that prioritizes user needs, accessibility, and modern interaction patterns throughout the activity feed implementation.
