üìñ Quick Usage Guide

Change Animal Status:

  1. Navigate to /admin/animals
  2. Find the animal in the table
  3. Click its status dropdown
  4. Select new status (Available/Adopted/Fostered)
  5. ‚ú® Done! Change is saved automatically

Move Animal to Different Group:

  1. Navigate to /admin/animals
  2. Find the animal in the table
  3. Click its group dropdown
  4. Select target group
  5. ‚ú® Done! Animal moved automatically

Export Comment Data:

  1. Navigate to /admin/animals
  2. (Optional) Select specific group to filter
  3. Click "Export Comments" button
  4. ‚ú® CSV file downloads automatically
  5. Open in Excel or any CSV viewer
